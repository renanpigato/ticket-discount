
## Requirements
[NodeJS](https://nodejs.org/en/)

## Running
```
npm install
npm run test
```

- Estudantes: R$ 8.00

      - 10% para todos(crianças, idosos, estudantes)

      - 5% estudantes;

      - 40% idosos
      - 30% crianças
      - 50% estudantes

Quinta-Feira
      - 30% idosos e estudantes

Sexta-Feira
      - 11% crianças

Domingo/Sábados/Feriados
      - 5% Idosos

       possui para todos os dias da semana, exceto finais de semana,
